💡 还没有网址？选一种免费方式部署：
  1. GitHub Pages（推荐）：
    → 打开 github.com 注册 → 新建仓库 → 点击 Add file → Upload files → 把整个 bernoulli-survey 文件夹拖进去 → Settings → Pages → 选 main 分支 → Save → 等 1 分钟得到网址
  2. Netlify Drop（最快）：
    → 打开 app.netlify.com/drop → 把 bernoulli-survey 整个文件夹拖进去 → 立即得到网址

部署后把网址粘贴到上方输入框，即可生成二维码。
