🔍 算法说明

本生成器使用 mulberry32 确定性伪随机数生成器（PRNG），配合 Fisher-Yates 洗牌算法，确保给定种子下输出完全确定。

// 1. 定义每题各选项的目标人数分布
const targets = [[42,38,13,7],...];
// 2. 为每题构造选项池
const pools = targets.map(t => buildPool(t));
// 3. 用 mulberry32(seed) 对每题的池独立洗牌
pools.forEach(p => fisherYatesShuffle(p, rng));
// 4. 第 i 位受访者取每池第 i 个元素
